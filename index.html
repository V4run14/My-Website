Personal Projects

Mr. Gobble & the Ghosts


Designed using C++, it revolves around the character Mr. Gobble who runs, jumps and climbs ladders between platforms to defeat his enemies, the ghosts while making sure not to lose all his lives. The ghosts can eat Mr. Gobble but are vulnerable when he eats a pellet, which gets randomly rendered on the game screen. Finally, the game provides a game score depending on time taken, lives and ghosts eaten and stores highscores.

Song Downloader



A Python algorithm to perform Web Scrapping. Automates Google Chrome to browse to my music playlist on YouTube and downloads N number of songs from the list. There is also an option to provide a list of songs to be downloaded. Saves the time and effort required to do it manually.

Battleships


The classic Battleships game implemented on C++. This 2-player game involves the setting up of 5 ships of different sizes within the the 10X10 Warzone Grid by one player. The second player then attempts to sink the ships by providing coordinates for his missile inside the grid. The game incorporates in game sounds of explosions and missile-firings. The player with the best accuracy wins the game.
